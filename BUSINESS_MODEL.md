# Align Business Model

Align is a free, open-source, local-first app. Monetization should add convenience, workflow value, or expert help without making the core app worse for free users.

## Principles

- Keep the local app free and useful.
- Do not make public users depend on the maintainer's private backend.
- Avoid in-app paywalls until the free product is trusted.
- Start with low-cost digital products before hosted infrastructure.
- Keep paid content separate from private client data.

## First Paid Offer: Workflow Template Packs

Sell downloadable workflow packs for web designers, freelancers, and small agencies.

Recommended first products:

- `Web Designer Starter Pack`
- `Figma to WordPress Project Pack`
- `Client Feedback + Revision Pack`
- `Website Care Plan Pack`
- `Freelance Weekly Planning Pack`

Each pack should include:

- Align template JSON
- Markdown setup guide
- project and task structure
- notes and checklists
- suggested weekly planning flow

Sales platform:

- Start with Gumroad for simple downloads, coupons, and updates.
- Keep one free sample pack on GitHub as proof of value.

Pricing:

- Single pack: `$9-$19`
- Bundle: `$29-$49`
- Launch discount: optional

## Second Paid Offer: Customization Services

After template demand is proven, offer paid service packages.

Suggested packages:

- `Setup Call` (`$49-$99`): install Align, import templates, configure local-first backup workflow.
- `Workflow Customization` (`$199-$499`): custom project boards, task flows, notes, reports, and planning setup.
- `Agency Setup` (`$500+`): client handoff setup, share-link workflow, self-hosted sync guidance, and reusable agency templates.

Customization should be scoped as services, not hidden app features.

## Future Hosted Offer: Align Cloud

Align Cloud can be considered later if there is demand for managed sync and convenience.

Possible paid cloud features:

- managed Supabase sync
- backup history
- share links
- Google Calendar/Todo sync
- email reminders
- signed convenience installers

Do not build this until the open-source app and template business have clear demand.

## What Stays Free

- local-first web and desktop app
- projects, tasks, todos, calendar, notes, reports
- full backup/export/import
- self-hosting docs
- free sample workflow pack
- source code

## What Can Be Paid

- premium workflow packs
- custom workflow setup
- agency/client workflow consulting
- signed convenience builds
- hosted sync and reminder service later

## Launch Path

1. Publish the free sample template in the repo.
2. Create one polished paid pack.
3. Sell it on Gumroad.
4. Make a short demo video showing import and usage.
5. Collect feedback before building more paid infrastructure.
