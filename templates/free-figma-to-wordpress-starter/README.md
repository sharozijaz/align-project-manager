# Free Figma To WordPress Starter Template

This sample pack shows how Align can structure a small website redesign project from discovery through Figma design, WordPress build, QA, and handoff.

## Contents

- `align-template.json` - importable Align workspace backup.

The template includes:

- one sample project
- parent tasks and subtasks
- calendar planning events
- project notes
- one reusable discovery note
- a few starter resources

## How To Try It

1. Open Align.
2. Go to Settings > Data.
3. Export a full backup of your current workspace.
4. Click Import Backup.
5. Select `align-template.json`.
6. Confirm the import.

Importing replaces the current local workspace after Align saves a safety copy. Test this in a fresh browser profile or separate desktop test install if you do not want to touch your main workspace.

## How To Adapt It

- Rename `Sample Local Business Website`.
- Replace placeholder links and notes with real client context.
- Drag tasks into Calendar Month or Week view to plan the work.
- Update task dates and statuses as the project moves from Figma to WordPress.
- Keep private credentials in notes only if your local/cloud security setup is appropriate.

No Supabase, Google, email, or hosted setup is required.
